Popover Widget

Реализация на чистом JavaScript без jQuery

Попробуйте кликнуть на кнопки:

Как это работает?

  • ✅ Чистый JavaScript — без зависимостей
  • ✅ Позиционирование в пикселях (без translate/transform)
  • ✅ Закрывается по клику вне popover и по клавише Escape
  • ✅ Полная поддержка BEM-методологии
  • ✅ Покрытие тестами через Jest + JSDOM

Инициализация через data-атрибуты:

<button 
  data-toggle="popover"
  data-title="Заголовок"
  data-content="Текст подсказки"
>
  Кликни меня
</button>
